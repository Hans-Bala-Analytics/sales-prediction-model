# Sales Prediction Model

## 📌 Project Overview
This project involved building a predictive model to forecast sales 
performance across 759 firms, using multiple machine learning 
classification techniques. The model was designed to be interpretable, 
helping identify key factors that influence sales outcomes.

## 🎯 Objective
- Predict sales performance using historical firm-level data
- Identify the most important factors driving predictive accuracy
- Provide an interpretable model to support informed decision-making

## 🛠️ Tools & Skills Used
- Python
- Exploratory Data Analysis (EDA)
- Data Pre-processing
- Logistic Regression
- Linear Discriminant Analysis (LDA)
- Decision Tree

## 🔍 Approach
1. Performed data pre-processing to clean and prepare data for modeling
2. Conducted EDA to understand feature distributions and relationships
3. Built and compared multiple models: Logistic Regression, LDA, 
   and Decision Tree
4. Evaluated model performance and identified the top 5 attributes 
   contributing to predictive accuracy

## ✅ Outcome
Delivered a robust, interpretable predictive model that enables 
data-backed investment and business decisions by highlighting the 
key factors influencing sales performance.
